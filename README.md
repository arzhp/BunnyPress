# BunnyPress
由Tossy的web服务（日语）创建的WordPress模板“BunnyPress”的轻版本。快速响应、移动优先、结构化标记。在某种程度上，我可以使用来自主题定制器的设计。由亦是爱网络汉化 https://www.yishiai.com
